# M8: 移动端 + 生态

**目标**：移动端（Android/iOS）可用，系统稳定化。移动端原生壳复用 `client/` Vue 前端（通过 WebView）。

**可见产出**：手机上也能提交任务、查看 Agent 状态、处理冲突。

---

## 阶段 8.1 Android 客户端

**目标**：Jetpack Compose 实现的 Android 应用。

**验收任务**：
- [ ] Jetpack Compose 项目初始化
- [ ] WebSocket 连接服务器
- [ ] 任务提交界面
- [ ] Agent 对话流展示
- [ ] 冲突通知（系统通知 + 应用内弹窗）
- [ ] 任务列表和状态
- [ ] Markdown 渲染
- [ ] 后台保活（WebSocket 断线重连）

**验收实践**：
```
1. 安装 APK 到手机
2. 配置服务器地址
3. 提交任务
4. 预期：实时看到 Agent 对话流
5. 触发冲突
6. 预期：收到系统通知，可在应用内决策
```
